\section{Reproducibilidad}

El código y los resultados livianos se encuentran en el repositorio \url{\repositoryurl}. La estructura incluye:

\begin{itemize}
    \item notebooks de preparación, pilotos, lotes y análisis estadístico;
    \item código modular en \texttt{src/pinn\_hrf};
    \item configuraciones de la PINN;
    \item resultados sintéticos y reales;
    \item pruebas unitarias del modelo Balloon--Windkessel;
    \item código fuente de este informe.
\end{itemize}

Los archivos fMRI volumétricos, datos crudos HCP y modelos binarios no se versionan. Para verificar el modelo fisiológico se ejecuta:

\begin{verbatim}
python -m pytest
\end{verbatim}

La ejecución local validada produjo dos pruebas aprobadas.
