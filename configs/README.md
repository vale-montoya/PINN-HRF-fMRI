# Configuraciones

Almacena aquí archivos JSON o YAML con:

- semillas aleatorias;
- parámetros verdaderos de los experimentos sintéticos;
- límites fisiológicos;
- hiperparámetros de GLM, MLP y PINN;
- pesos de la función de pérdida;
- número de épocas;
- parámetros de Adam y L-BFGS;
- definición de escenarios y ROI.

Los notebooks deben leer estas configuraciones cuando sea posible, en lugar de duplicarlas en múltiples celdas.
